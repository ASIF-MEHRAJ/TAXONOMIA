# Taxonomia: Biological Specimen Landing Page

**Taxonomia** is a high-end, modern landing page designed to showcase the scientific classification and physical specifications of the *Pachyteria equestris* specimen. Built with a focus on minimalist UI/UX and semantic HTML, this project blends academic entomology with premium digital design.

## 🚀 Overview
The project serves as a digital museum archive, utilizing a split-screen visual identity to organize complex biological data into an elegant, user-friendly interface.

## ✨ Key Features
- **Split-Screen Layout:** Organized dual-pane containers for scientific taxonomy (left) and product specifications (right).
- **Interactive Toggles:** Custom-built CSS toggle switches for dynamic state changes (e.g., View detailed antennae).
- **Modern Typography:** High-contrast serif fonts used for biological nomenclature to establish an authoritative feel.
- **Specimen Visualization:** A centered 3D render of the specimen housed within a glass cloche.
- **Clean Architecture:** Semantic HTML5 and modular CSS Flexbox layout for easy scalability.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom animations, Flexbox layout, and glassmorphism effects.
- **Remix Icons:** Minimalist iconography for data points.

## 📁 Project Structure
- `index.html`: Main structural markup.
- `style.css`: Custom styling and layout logic.
- images are not in the folder,so you can directly copy paste the code
